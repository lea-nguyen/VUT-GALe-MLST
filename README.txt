Execute the jar with :
	java -cp .\mlst.jar maximumLeafSpanningTree.Graph <integer> <|"2-app"|"3-app">

Explanation of arguments:
1st
	integer : |V| of the graph to create
2nd
	no-argument : test both algorithms
	2-app : 2-approximation algorithm to execute
	3-app : 3-approximation algorithm to execute
